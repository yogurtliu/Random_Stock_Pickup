# 股票分析程序

一个基于React的股票分析工具，支持导入Excel股票数据，随机选择股票，并获取AI分析报告。

## 功能特性

1. **Excel文件导入** - 支持导入包含指定列的Excel文件，自动验证格式
2. **随机股票选择** - 从导入的数据中随机选择2支股票
3. **AI分析集成** - 将选中的股票数据提交至AI分析接口获取专业报告
4. **结果展示** - 以结构化方式展示分析结果，包括基本信息、财务指标、市场表现和投资建议

## 必填列

Excel文件必须包含以下列：
- 代码
- 名称
- 所属概念
- 细分行业
- 所属同花顺行业
- 总市值
- 主力净量
- 市盈(动)

## 安装

```bash
npm install
```

## 运行

```bash
npm run dev
```

## 构建

```bash
npm run build
```

## 配置

如需使用真实的AI分析接口，请创建 `.env` 文件并配置：

```
VITE_API_BASE_URL=你的API地址
```

默认使用模拟数据，无需配置API即可体验完整功能。

## 技术栈

- React 18
- Vite
- xlsx (Excel解析)
- axios (HTTP请求)
